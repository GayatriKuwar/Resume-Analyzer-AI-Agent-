# AI Resume Score Analyzer

## 📋 Project Overview
An AI-powered resume analysis tool that provides instant feedback, scoring, and improvement suggestions.

## 🚀 Features
- PDF resume upload
- AI-powered analysis using OpenAI
- Overall score with detailed breakdown
- Strengths and weaknesses identification
- Job role-specific suggestions
- Interactive web interface

## 🛠️ Technologies Used
- **Frontend:** Streamlit
- **AI/ML:** OpenAI GPT-3.5
- **PDF Processing:** PyPDF2
- **Language:** Python 3.14+
